# Flera dimensionskolumner i frihandstabeller

Gör dina frihandstabeller ännu kraftfullare med funktionen Flera Dimension Columns i CJA. Förutom standarddimensionsuppdelningar kan du analysera dimensionsdata sida vid sida för att skapa relationspar som bättre motsvarar och förstår dina data. Den avancerade sorteringsfunktionen ger också större flexibilitet att hantera tabeller så att data visas precis som du vill!
