# Realtidsrapportering

Läs om hur realtidsrapportering i Adobe Customer Journey Analytics ger viktiga mätvärden som intäkter, konverteringar och besök direkt i Analysis Workspace. Idealiskt för större kampanjer, produktlanseringar och resevalidering - med den här funktionen kan marknadsförarna fatta snabbare, datadrivna beslut varje minut.

Mer information finns i [dokumentationen](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-components/real-time-reporting/real-time){target="_blank"}.
